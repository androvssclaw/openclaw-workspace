# VPN Anti-Block — Quick Card

1) **Проверка**
```bash
./scripts/vpn_status.sh
openclaw status
```

2) **Fallback-профиль**
- Переключись на резервный профиль.
- Тест 1–2 минуты.

3) **Смена UDP-порта**
- Выбери новый порт (до 9999, свободный).
- Обнови AWG и пересоздай клиентский конфиг.

4) **Смена ключей/конфигов**
- Выпусти новые ключи/конфиги.
- Старые отзови.

5) **Смена обфускации**
- Обнови I-параметры/сигнатуру.
- Снова выпусти конфиги.

## Правила
- Меняй по 1 параметру за итерацию.
- firewall/docker/systemd/сеть — только после подтверждения Andrey.
- Держи 2+ резервных профиля заранее.
